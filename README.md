
**config配置文件说明**

username=xxx

登录帐号xxx替换为6位以上的帐号，不存在的帐号会自动创建

password=xxx

登录密码xxx替换为8位以上的数字字母

username12306=xxx

xxx替换为12306登录的帐号名

password12306=xxx

xxx替换为12306登录的密码

email=xxxx

xxxx替换为接收抢票成功的通知邮箱地址

fromstation=广州南 

出发站点

tostation=桂平

到站站点

date=2017-01-18

乘车日期,格式为xxxx-xx-xx月份跟日10一下需要补前导0
